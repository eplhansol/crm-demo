# 영업 관리 CRM (CRM DEMO)

영업 파이프라인·고객·거래처·할 일을 한곳에서 관리하는 **영업 CRM** 웹 애플리케이션입니다.
칸반 파이프라인, 거래/고객/거래처 관리, 할 일(후속 조치), 대시보드, 문의 접수 → 리드 전환,
엑셀 가져오기/내보내기, 역할·권한 관리를 제공합니다.

🔗 **데모 사이트: https://eplhansol.github.io/crm-demo/**
(데모는 가상 데이터로 동작하는 프런트엔드 단일 페이지입니다. 접속하면 자동 로그인되어 바로 둘러볼 수 있습니다. 실제 운영본은 사내에서 비공개로 운영 중입니다.)

## ✨ 주요 기능

- **파이프라인(칸반)**: 거래를 단계(리드·접촉·제안·협상·수주·실패)별로 보고 드래그로 단계 변경
- **거래(딜) 관리**: 금액·수주확률·담당 부서·종결 사유, 거래별 활동(통화·미팅·메일·메모) 기록
- **고객/연락처**: 고객 구분(잠재·일반·핵심·휴면), 부서·직책, 고객 메모, 검색·페이지네이션
- **거래처**: 업종·매출·영업상태, 최근 7일 신규 NEW 배지, 등록일 정렬
- **할 일**: 후속 조치(지연·오늘·예정·완료) 한눈에 보기
- **문의 접수 → 리드 전환**: 회사·고객 중복 판정 후 재사용, 새 리드 자동 생성
- **대시보드**: 단계별·담당자별 현황, 전환율·가중 예상매출
- **사용자/그룹·권한**: 관리자/매니저/멤버 역할, 멤버 세분 권한(편집·삭제·내보내기)
- **엑셀 가져오기/내보내기**: 중복은 업데이트(upsert), 신규/업데이트 건수 안내

## 🛠 기술 스택

- **React 19 + TypeScript**
- **Vite** (빌드/번들)
- **React Router**
- 데모 모드: 인메모리 + `localStorage` (백엔드 없이 동작)
- 실서비스 모드: **Supabase** (Postgres + Auth)

## 📌 참고

- 이 저장소의 `index.html`은 데모용으로 빌드된 **단일 페이지**입니다.
- 데모의 회사·인물·연락처 데이터는 모두 **가상**이며 실제 기업/개인과 무관합니다.
- 비밀 키·실데이터는 포함되어 있지 않습니다.
